# AI漫剧提示词输出契约

Use this reference when generating complete AI video prompts, 15-second segments, shot lists, image prompts, or nine-grid storyboard prompts.

## Complete 15-Second Segment

Use this structure for each independent segment:

```text
[分镜编号]：XX｜标题｜最终版

黄金前3秒：
[一句强视觉钩子，明确核心冲突、动作或情绪爆点。]（时长：约15s）

[人物]：
[本段出场人物，重点人物最多3-4个。其他人只作为虚化背景、离焦轮廓、肩膀、手部或反光处理。]

角色A：
[性别、年龄段、外貌、服装、当前状态、情绪、动作倾向。]

角色B：
[性别/类型、外貌、材质、服装/机体结构、当前状态、情绪或系统状态。]

背景人物/群体：
[如有，只能作为虚化背影、离焦轮廓、黑屏阵列、肩膀、手部或反光处理。]

[人物的详细站位]：
本段所有方位均以观众看到的画面为准：画面左=屏幕左侧；画面右=屏幕右侧；画面上=屏幕上方；画面下=屏幕下方；画面前景=靠近镜头；画面中景=人物主要动作区域；画面后景=远离镜头。

START_POSE：
[第一帧人物站位/姿势：谁在画面左/右/前/后；距离镜头约几米；谁面对谁；谁背靠什么；谁抓住什么；谁悬浮/站立/跪地/坐着；关键道具在哪里。]

动作坐标：
1. [人物A从哪里移动到哪里。]
2. [人物B保持在哪里，或被什么力量带到哪里。]
3. [门、按钮、武器、裂缝、屏幕、车辆、高台等锚点固定在哪里。]
4. [动作接触点在哪里。]
5. [动作结果落点在哪里。]

END_POSE：
[最后一帧空间关系，并为下一段留下眼神、动作、道具变化、能量变化、空间压迫或危险预兆。]

[场景空间]：
LOC：[地点锚点，内景/外景，时间。]
空间结构：[几何结构、纵深、入口出口、墙体、天花板、地面、高台、门、走廊、平台、窗、栏杆、机器、屏幕。]
ATMOSPHERIC_LAYER：[雨、雾、尘埃、体积光、霓虹、烟气、火花、漂浮碎屑、失重微尘。]
ENV_EMOTION：[环境情绪，例如权力压迫、末世孤独、危机逼近、神秘悬疑、史诗震撼。]
ERA_ANCHOR：[近未来、古代王朝、民国、现代都市、赛博都会、废弃空间站等。]
RENDER_STYLE：[3D动漫质感 / 3D游戏CG / UE5渲染 / PBR材质 / 电影级虚拟摄影 / 真人写实 / 好莱坞级电影视效。]

[人物、动作与台词]：
* 【[焦段mm / 机位 / 景别]｜[专业运镜术语]｜[时间段]】
  [具体画面描述：人物当前站位、动作起点、动作过程、动作落点、前景遮挡、表情变化、视线方向、与上一镜/下一镜的动作承接、光影如何落在脸/手/道具/空间锚点上。]
* 角色名（微表情/眼神/神态/情绪/气口/停顿/重音/句尾落点/呼吸节奏）：“严格复制原台词，一字不改”
```

Repeat 5-8 shots as needed. The final shot must leave a clear next-segment hook.

## Required Technical Modules

After the shot list, include:

- `[镜头语言]`：说明开场如何抓人、中段如何升级、动作如何承接、视线如何引导、反应镜头如何释放情绪、结尾如何形成钩子。
- `[视觉特效/UI]`：粒子、流体、体积雾、布料解算、能量涟漪、环境破碎、微尘悬浮、全息UI、屏幕UI、镜面反射、空间扭曲、火花、碎片。UI文字只能存在于设备屏幕内部。
- `[光影氛围]`：主光、辅光、轮廓光、负补光、动态光源、色彩情绪；必要时写明色温。
- `[声音设计]`：环境声、动作声、设备/特效声、对白声线、音乐要求。
- `[互动锁定]`：人物位置、距离、朝向、接触点、道具位置、动作顺序、禁止穿模/融合/瞬移/换位/翻轴。
- `[连续性技术锁]`：角色造型、场景锚点、摄影轴线、动作方向、光色、台词顺序、物理规则、结尾可接续姿势。
- `[关键帧]`：KEYFRAME_START、KEYFRAME_MID、KEYFRAME_END。
- `[负面约束]`：无外挂字幕、水印、logo、旁白文字、多余人物、换位、翻轴、瞬移、穿模、肢体融合、多手多脚、手指畸形、面部崩坏、表情漂移、动作重复、镜头乱晃、背景穿帮、过曝、低清、塑料皮肤、塑料金属、AI感、画面外UI。
- `[节奏描述]`：前3秒钩子、中段加速、后段落点、结尾钩子。
- `[风格]`：优先使用用户指定风格；未指定时使用技能默认风格。

## Nine-Grid Storyboard

Use when the user asks for 九宫格、分镜图、等比例多宫格、站位图、image storyboard, or keyframe grid.

```text
等比例九宫格分镜头｜[分镜编号]

规格：3x3九宫格；所有画格等比例；单格建议16:9；画面方位必须和视频提示词一致；每格都是一个独立关键镜头；用于AI生图 / 分镜参考 / 视频站位参考。

[九宫格 01｜标题]
景别：[焦段 + 景别]
画面：[人物位置、场景锚点、动作状态。]
站位：[谁在画面左/右/前/后，谁靠近什么，谁面对谁。]
重点：[建立位置、确认落点、反应、接触、拉离、格挡、收束等。]
```

Repeat through `[九宫格 09｜标题]`.

统一约束：比例一致、同一空间坐标、同一摄影轴线、不翻轴、角色和道具位置固定、动作可连续剪辑、重点人物最多3-4个、背景人物只做虚化轮廓或静态背景、画面清楚。
