# AI漫剧质量评分卡

Use this reference when the user asks for analysis, scoring, diagnosis, "哪里不对", "能打几分", "9.6分以上", or optimization advice.

## Scoring Rule

Score honestly. Do not flatter weak prompts. If the result is below 9.6, name concrete failure points and provide executable fixes.

## Review Angles

Evaluate at least these angles:

1. 剧本忠诚度：台词是否完整、不改字、不乱序。
2. 分段独立性：每段是否重新建立人物、空间、动作、光影。
3. 镜头衔接：上一镜和下一镜是否能自然剪接。
4. 段落钩子：上一段结尾和下一段开头是否好接。
5. 人物站位稳定性：是否存在换位、瞬移或空间漂移。
6. 物理坐标清晰度：左/右/前/后/高/低/远/近/朝向/接触点是否明确。
7. 动作物理：力、惯性、反作用、落点是否可信。
8. 情绪递进：气口、停顿、重音、眼神、微表情是否服务剧情。
9. 电影张力：镜头、构图、节奏、黄金前3秒是否有冲击。
10. 光影质量：主光、辅光、轮廓光、负补光、动态光源、色温是否连贯。
11. 导演意图：构图和视线是否有叙事目的。
12. 视效服务度：特效是否服务剧情，而不是堆砌。
13. AI穿帮风险：负面约束和技术锁是否覆盖常见生成失败。
14. 时长适配：当前剧情是否适合一个15秒段落，或必须拆成多段。
15. 最终评分：是否达到9.6+。

## Response Shape

```text
最大问题：
1. [具体问题 + 为什么影响生成。]
2. [具体问题 + 为什么影响连续性。]
3. [具体问题 + 为什么影响情绪或视觉冲击。]

可执行优化：
1. [具体改写指令。]
2. [具体连续性或坐标锁。]
3. [具体镜头/光影/表演强化方案。]

是否达到9.6+：[是/否。如果否，说明必须补齐什么。]
```
